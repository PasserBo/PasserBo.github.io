About me

Extensive Experience in Machine Learning

I have extensive experience in machine learning through various ML projects, where I have developed and fine-tuned models to meet specific requirements.

  • Research Assistant: During my experience as a research assistant in the Department of Biomedical Engineering at Case Western Reserve University, I implemented machine learning algorithms for data analysis and predictive modeling.

  • Transfer Learning: I worked on the Domain Adaptation for Visual Emotion Classification project, where I applied transfer learning techniques to improve model performance. These experiences have enhanced my ability to adapt to new machine learning tasks and apply appropriate methodologies based on project
    needs. [Code](GitHub - JackyXu866/Circular-structured-representation: This is the official implementation of 2021 CVPR paper "A Circular-structured Representation for Visual Emotion Distribution Learning".)

Proficiency in Data Analysis

My knowledge extends to various aspects of data analysis, including text and image analysis.

  • Text Analysis: As a developer of a Large Language Model Text Summarizer Application, I integrated the GPT-3.5-turbo model to create a web app that effectively condensed large-scale texts into more manageable formats. This hands-on experience has equipped me with practical skills in developing and utilizing advanced text analysis methods.

  • Image Analysis: I have experience in image analysis and computer vision. I have worked on projects involving image classification, object detection, and segmentation using convolutional neural networks (CNNs) and frameworks such as TensorFlow and Keras. For instance, in a project on visual emotion classification, I utilized CNNs to extract features from images and classify them based on emotional content.

Advanced Python Knowledge and Common Data Analysis Libraries

  • Through numerous research projects and coursework, I have honed my skills in Python and essential data analysis libraries such as pandas, NumPy, NLTK, gensim, Matplotlib, and scikit-learn. Additionally, I have considerable experience using JupyterLab and Google Colab to facilitate data analysis and visualization tasks efficiently.
